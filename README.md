# Mini-Supermarché – Caisse Management (C)

This project is a C program that simulates the management of a mini-supermarket checkout system.
It was developed as a student project to practice programming in C and the use of data structures.

## Project features
- Product management (add, search, display)
- Client management
- Checkout simulation
- Simple menu-based interaction
- Use of classic data structures (linked lists, queue, stack, tree, hashing where applicable)

## Technologies
- Language: C
- Compiler: GCC

## How to run the program
```bash
gcc -o app main.c
./app
